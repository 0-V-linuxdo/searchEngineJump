# searchEngineJump fork

本 fork 基于qxinGitHub/searchEngineJump 的5.26.11 版本

## 相对原分支的修改

| 场景 | 用户可见变化 |
| --- | --- |
| 搜索页支持 | 在 Kagi 搜索结果页也能显示搜索跳转栏；YouTube 新版结果页和 bilibili 全部搜索页显示更正常；进入 YouTube 视频播放页时不再插入跳转栏。 |
| 暗色页面 | 在深色主题页面中，跳转栏、下拉菜单和设置窗口使用更稳的深色背景，减少透明发亮、边缘突兀和文字不清的问题；Kagi 搜索页上的跳转栏也更贴近页面布局。 |
| 图标显示 | 搜索引擎图标会缓存在本地，Kagi 等限制较严格的页面也能正常显示图标。 |
| 设置窗口 | 设置窗口会跟随页面明暗主题，打开和滚动时不再容易穿透页面、挤出屏幕或错位。 |
| 设置入口 | 即使把设置按钮调到完全透明，也仍能通过鼠标悬停找回并打开设置。 |
