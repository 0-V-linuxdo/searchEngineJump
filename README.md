# searchEngineJump fork 修改说明

本分支基于原项目 `qxinGitHub/searchEngineJump` 的 `5.26.11` 版本维护，只记录 fork 相对原分支做过的修改。

## 修改内容

- 适配 YouTube 新 UI，并排除 YouTube 视频播放页，避免在视频页注入跳转栏。
- 适配 bilibili 全部搜索结果页。
- 新增 Kagi 搜索结果页适配。
- 修复 Kagi 暗色模式下脚本按钮栏与设置弹窗的样式问题。
- 优化设置弹窗，跟随页面明暗主题，减少页面样式污染。
- 修复设置弹窗滚动穿透、窄屏溢出和布局错位问题。
- 修复设置按钮透明度为 0 时无法通过 hover 唤出的问题。
- 优化脚本按钮栏下拉菜单在暗色模式下的背景，避免右侧出现透明亮色。
- 整理脚本元信息与 `@update-log`，当前版本为 `[20260504] v1.1.2`。
